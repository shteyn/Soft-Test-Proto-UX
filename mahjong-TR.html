Rewarded Referrals - In Session

Description:

Users are able to send referrals to other users. Then when the referred user joins the game the user is rewarded with coins. Reward requirements: The user that is referred must have never played our game before Host user can receive rewards in and out of session

Test Prerequisite:

2 new users in FB that are friends with each other (User A and User B)

Steps:

1. With User A Start the game.
2. After FTUE click on the referral icon on the map screen
3. Click on invite on the new popup
4. Select in messenger and then select User B to send the Invite to.
5. Do not close User As game.
6. With User B goto FB messenger and click on the link given in User A/B’s chat this will open our game
7. Back to User As game the referral icon should have a exclamation point - click on it.
8. Observe that User Bs avatar is now listed and you can claim the coin reward
9. Check that User A got the coins
